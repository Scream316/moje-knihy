📝 Jak přidat knihu

Knihy se přidají manuálně úpravou souboru books.json v GitHub repozitáři.

Příklad JSON formátu:

{
  "knihy": [
    {
      "nazev": "Název knihy",
      "autor": "Autor knihy",
      "dobrovskyUrl": "https://dobrovsky.cz/product/xxx",
      "obalkaUrl": "https://example.com/cover.jpg"
    }
  ]
}

⚠️ Pozn: obalkaUrl je nepovinné. Pokud ho nevyplníš, použije se placeholder obálka.
